论大A股的有效性

发表于 2019-02-28 | 分类于 金融漫谈 |

年后这两天，金融市场的主角无疑是我们的大A股，2月25日，上证指数暴涨5%，券商全面涨停。

这剧情是不是十分的熟悉？

我赶紧打开多年不用的同花顺看看是不是有什么新的变化。但翻着那些熟悉而又尴尬的股票，我发现A股现在确实还没有一点新的东西：妖股东方通信和多年前的特例A、恒立实业等等一样，靠游资带节奏翻了10倍；券商股一有什么风吹草动就涨停，而且特别整齐；钢铁、医药、通信板块轮番带节奏，最后由白酒股收官，同样的味道在08、15年都不知尝过多少次了。。。

看了一圈，我依然可以得出一个结论：由于畸形的涨跌停、上市退市的审批制度，A股目前依然是资金市、消息市、政策市、情绪市，依然没有一个成熟市场应有的稳重，换句话说，还是那个割韭菜的市场，广大股民依然是用玩赌场的心态来炒股。

这让我想起了一个词：市场的有效性。究竟何为市场的有效性呢？尼古拉斯·塔勒布曾经在他的著作《黑天鹅》中说过：一个市场，不管是在哪个国家，其日常的交易有99%都是噪音和杂点，只有1%是真正有效的（这里主要指的是交易量）。也就是说，真正决定市场未来走向的交易只占整个市场的很小一部分，如果一个市场趋于理性，那大部分噪音交易的结果会将是互相抵消，有效交易短期内有可能没有大的影响，但长期下来影响的效果还是显著的，那这个市场长期来看就是有效的。但问题是，A股市场上的大部分交易者是散户，缺乏理性，有着显著的羊群效应和追涨杀跌心理，表现出来就是典型的旅鼠效应，市场就会暴涨暴跌，长期无法呈现正常的估值，大量垃圾股充斥市场、价格畸高。典型的例子就是中毅达（*ST 毅达，证券代码：600610）

这种奇葩情况估计只有我们大A股才会出现吧！你说在这样的市场投资，你能有多好的收益呢？

最近科创板即将闪亮登场，政策上放宽了不少，如每日20%涨跌幅、IPO由审批制改为注册制、上市没有盈利要求和股权类比要求（也就是说AB股类型的公司也可上市）、上市前五日不设涨跌幅等等。可以看出，监管层也知道目前A股的弊端，想要改革一下了。至于效果如何，能够吸引到多少优秀企业上市，我们就拭目以待吧。

目前大家的当务之急，是要想办法把科创板权限开通，毕竟从以上分析来看，只有科创板才有比较大的概率能成为一个有效的市场，你们所学的巴菲特的那套东西（价值投资？？）才能真正派上用场。其他的？呵呵o(￣︶￣)o
