品《读库1803》

发表于 2019-02-01 | 分类于 读书笔记 |

前两天，我把《读库1803》读完了，阅读的总体感觉就一个字：爽！读完这本书我终于体验到，如果你的阅读达到一定量的时候，许多你读到的文章会让你会想起之前阅读过的文章，新的知识点与原有的知识点相互交织，从一个个孤立的点逐渐编织成一张网，从而让知识丰富而立体了起来，这是一种无与伦比的体验。

首篇文章是对作家毕飞宇的采访稿，重点讲述了他的人生轨迹和创作历程，以他创作最重要的几篇长篇小说为脉络，告诉了大家他是如何成为一名作家和自己对写作的感悟。这篇文章同我之前所阅读的斯蒂芬金的《写作这回事》、村上春树的《我的职业是小说家》均为同一主题，就是作家谈自己的写作。虽然这几位作家国籍、背景、风格均大不相同，但他们的写作经历均有相似之处：都是从青年甚至是少年时代起便勤于写作、在处女作发表之前均遭遇过无数次退稿（村上春树相对会少一些，他是个幸运儿），但都百折不挠，终获成功。他们都很注重发现生活中的美、记录生活中的各种人和事（并不是所有时候都用笔记录，大部分还是用心去记录）。而毕飞宇则重点强调了几点：一是要注重在少年时期就培养审美，这其中既包括欣赏大自然的美，也包括欣赏其他作家、尤其是中国古代文豪的优秀文学作品。他说：欣赏优秀的古文会让你建立起一种语感，当你的阅读和体会达到一定量以后，你的文章仿佛天生一般，不需要刻意模仿便会自带一种韵律，仿佛诗词一般优雅。毕飞宇自己的文章就有这个特点，文字风格既优雅又简洁，这与他的父母从小给他大量的阅读，尤其是古代优秀文学有着很大的关系。二是要保持好奇心，只有对生活充满好奇，不断记录生活的点滴，作家的创作源泉才不会枯竭。这一点上，毕飞宇和斯蒂芬金还有村上春树的观点极为相似。其实，作家绝大多数时候和我们是一样的，他们创作的灵感多来自生活，只是他们通过艺术的手法将其转化升华，形成一个个精彩的故事呈现给了读者。他们与我们最大的区别就在于一双慧眼和不断产生的好奇心。第三，毕飞宇还提到，他认为一个优秀作家不光要有美感，更重要的是要有对时代、文化的批判精神，只有在作品中展现文字美的同时体现出这种人文关怀，才称得上是大作家、优秀作家。举个例子，在他心中，汪曾祺不算是优秀作家，他的作品很美，但缺乏人文关怀、缺乏对世界的批判；鲁迅则是代表近现代做到这2点作家的最高峰，在他心中，鲁迅远比汪曾祺、周作人等人优秀。

第二篇文章《与深渊同行》，刊发了作者所画的简笔画和画中所表达的种种人生百态，这些人生重点集中在北漂群体，他用简洁、平静的笔触表达了北漂们的孤独、苦闷和对生活的向往和热爱，不带有任何立场，却又不得不让人陷入沉思，他描绘的，不就是生活中我自己的影子嘛。

第三篇，《从工厂到大学》，描述了作者从一个普通工人自考到重点高校读研的不平凡经历。其中最让我触动的，就是作者对华中科技大学图书馆的详细描写，非常仔细，让读者仿佛亲临现场，也让我十分向往武汉这座城市的文化氛围，东莞何时能够成为第二个武汉呀？

第四篇，《中考魔方》，讲述了作者陪伴孩子历尽艰辛考取高中的故事。从郑州一个普通家长的视角告诉了我们河南省学生竞争有多么激烈。有意思的是，作者在文章里列举了一道中考数学题，十分有意思：

学校“百变魔方”社团准备购买A、B两种魔方，已知购买2个A种魔方和6个B种魔方共需130元，购买3个A种魔方和4个B种魔方所需款数相同。

（1）求着两种魔方的单价；

（2）结合社员们的需求，社团决定购买A、B两种魔方共100个（其中A种魔方不超过50个）。某商店有两种优惠活动。请根据以上信息，说明选择哪种优惠活动购买魔方更实惠。

活动一：“疯狂打折”，A种魔方八折，B种魔方四折；
活动二：“买一送一”，购买一个A种魔方，送一个B种魔方。

这道题难度并不太大，我自己当时看到以后用心算就解出了第一问。

首先用数学语言翻译一下题干：2×A+6×B=130，3×A=4×B。则6×A=8×B，即6×A+18×B=390，则（8+18）×B=390，算出B=15，A=20；

第二问：分别将活动一和活动二的最优惠方案算出再进行比较即可，将题干用数学语言表达：A+B=100（其中0≤A≤50），则很容易算出活动一的最优方案：A=0，B=100，共花费：15×0.4×100=600；活动二的最优方案：A=50，B=50，共花费20×50=1000。所以活动一更优惠。

不过，这道题令广大考生奔溃的是，有部分学生和家长将题干理解成了另外一个意思：买3个A和4个B所需价格也是130元，而且，从这个意思去理解，题干也能说通，这样题目就出现另外一种答案了。只能说，中国语言博大精深，出题者不够严谨，实际上只需要把题干2句话调转顺序就不会产生歧义了。

但问题既然出现，后面的情况就一发不可收拾了，剧情发生了几次翻转：一开始省教育厅承认有歧义，两种解法都能给分，但马上有人发现，就算按照第一种解法（也就是我的解法），给出的第二问的参考答案依然有误。也就是说，没按照正常理解的同学可以拿到满分，反倒是按照正常理解，完全做对考题的学生反而不能拿到满分。这个结果令人咂舌，只能说，出题组简直把十年寒窗的学子的前途命运当作儿戏！当然，不管家长怎么闹，怎么抗议，最终的结果，依然是不了了之。

这篇文章让我心情很压抑，我也有两个小孩，如果是我遇到这种情况，或许也只能是默默接受吧。其中的道理你我都懂，虽然很难过，但这就是中国的现状。

第五篇，《黎曼假设平话》，这几乎是我最喜欢的一篇。作者用深入浅出的语言，从最简单的素数知识不断引申，最终解释了什么是20世纪乃至21世纪人类未解的数学难题之首：黎曼假设。我看完后，有7、8成懂，其中最让我感到奇妙的就是欧拉恒等式的美妙之处：

eiπ + 1 = 0

他巧妙的将自然对数e、圆周率π、虚数i和实数0、1用最简洁的方式结合在了一起。我不由得想起了卡尔·萨根在他唯一的科幻小说《接触》中所写的：在超越数π中蕴藏着造物主的密码，这是我们所在的宇宙中共同的语言。而恰恰在欧拉恒等式中存在着两个超越数：e和π，这难道仅仅是巧合？
