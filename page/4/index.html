读《遗落的南境》系列

标签（空格分隔）： 读书笔记

《遗落的南境》系列一共有3本（名字分别叫《湮灭》、《当权者》、《接纳》），是美国科幻作家杰夫•范德米尔在2014年起陆续写就的杰作。其中的第一部《湮灭》打败了当年的大热门《三体》获得2014年度星云奖。

读完这三部书以后，我的总体感受是：晦涩难懂，但又不得不承认作者成功的让读者浸入到了他所营造的那种灰暗、神秘的气氛中，久久不能自拔。尤其是作者在三部曲中，不断的重复着：

“出自罪孽者之手的扼杀之果既已在此我将孕育出死亡的种籽与蠕虫分享……”

仿佛是一段咒语，让读者眼前呈现出南境的那种湿热、沉闷、压抑的环境，同时也让我们体验到主人公内心的那种恐慌和不安。相当有画面感。

《遗落的南境》三部曲情节比较曲折，虽然三部分别都讲述的是独立的故事，但读者能清晰的体会到其中的脉络。而且书中所营造的科幻场景是作者所原创的（虽然几乎可以将此书归类为奇幻志怪小说），对于神秘的“X区域”的描写十分富有想象力，从而可以让每一个读者在自己的脑海里构架出一个自己的“X区域”。在此区域，人非人、物非物、鸟非鸟、兽非兽，一切事物都大不寻常，但你又说不清到底是什么原因。如果你进入了这个区域，你将不再是“你”（或者说在别人的视角中不再是了），你将成为“X区域”整体中的一部分而存在，你虽有独立意志，但也有一部分意志受“X区域”支配。这是多么天马行空的想象！

也许想象力的驰骋，就是我喜欢读科幻小说的原因。或许你也可以试试。

武汉学习见闻

发表于 2018-09-23 | 分类于 游记 |

上周有幸在武汉大学学习培训了一周，对武汉和武大有了一些粗浅的印象。总体感觉：武汉是个有朝气的准一线城市。武汉大学真大！

我们头一天逛了长江夜景、走了一圈户部巷的小吃街，虽然长江夜景没有珠江那么繁华，但晚上十点半过后，武汉人依然活跃在大街小巷的场景让我吃惊，因为在东莞这十年所感受就是：大家过了晚上十点就不敢在街上乱走了，尤其是小姑娘。起码武汉的治安是肯定要比东莞好的。

这一周，武汉大学没少逛，虽然很不凑巧，校史馆、周恩来故居均没有开放，但我还是深入游览了万林艺术博物馆和校内其他的一些景色。尤其万林艺术博物馆给我印象深刻，让我参观了世界各地的奇观异景（均为遥感拍摄），还有武汉有名的青年艺术家的画作，涨了见识、开了眼界。

等我小孩大一点，我也要带她来感受一下真正的“大”学是什么样子的！

现在CPU的套路如出一辙, 各位韭菜就不能长点记性吗?如果真想在EOS上面玩金融,那干脆咱们彻底一点,全面开放RAM现货、期货、租赁市场；全面开放cpu、带宽现货、期货、租赁市场；全面放开所有EOS代币的现货、期货市场，看看能够发生什么？彻底把中心化交易所给干掉！

BM,记住,DAPP开发才是正道,记住长者的话:”房子是用来住的,不是用来炒的!”

一句话自我介绍

大家好，我是尼克拉斯布鲁，一个低调而不奢华，胆小却爱金融的终身学习者。

一分钟介绍自己

我是一个热爱金融行业，喜欢计算机技术的一名小小公务员。已步入而立之年的我从5年前开始炒股、2年前开始炒币，就与金融和区块链技术结下了不解之缘。为了改变自己的命运，我一直践行着终身学习的理念，不放过任何一个学习、提高自己的机会。2个月前，我加入了007这个大家庭，就是因为在这里有一群和我一样爱学习，但却比我还要努力的战友和我并肩战斗，让我找到了组织。我希望在这个大家庭里早日成长，和大家一起打造属于自己的品牌，早日实现财富自由！

三分钟推销自己

我从小就对计算机技术十分感兴趣，虽然工作以后一直从事着与专业无关的岗位，但自己从来不敢放下专业技能，一直想着有朝一日又能派上用场。2013年，工作4年的我有了一点小积蓄，便想着投入股市。因此我读了很多有关股票投资的书籍，也报了许多培训班。收获很大，但最大的收获是在我入市以后在实践中不断的思考和总结。到了2017年，我在股市上的年化收益均在20%以上（2015年超过150%）。

就在2017年春节前夕，一个朋友向我介绍起了比特币，说他一个月翻了10倍。这顿时引起了我的注意。我花了3天时间翻出了有关比特币的所有资料进行阅读和思考。深深地被比特币这种去中心化的模式所吸引，顿时决定全仓入市，至今一分未动。虽然经历了94和之后的几次暴涨暴跌，我的心态反而越来越平和，深陷于区块链的魅力中而不能自拔。不断的研究和投资各种币种，并从今年起尝试自己写专栏文章介绍区块链，并帮朋友打理一些资金。体会就是：币圈一日，人间一年。说的就是你在这个圈子里的成长和变化速度非常之快，让你真真正正的有种和时间赛跑的感觉。

今天覃老大也和我们分享了他进入币圈的故事，我希望每一个战友，不管你有没有投资区块链，首先你要知道和了解这项技术，否则你就会被这个时代所淘汰。欢迎大家和我一起成长进步！

反击的号角

发表于 2018-09-15 | 分类于 区块链 |

前两天，ETH跌到了前所未有的低点：165刀，随后，随着一个假消息（比特币ETF被审核通过！？），ETH、BTC和EOS开始绝地反击，比特币和EOS反弹了大概10%左右；ETH反弹最为猛烈，从165刀直接涨到220刀，一举收复前期失地。

按照市场的惯例，这次反弹大家又在找各种原因来解释，普遍的说法是：纽约金融管理局批准2个基于eth的锚定美元的稳定币：GUSD和PAX，按照肖磊的说法是：

2018年9月10日，纽约金融服务部（NYDFS）同时批准了两种基于以太坊发行的稳定币，分别是 Gemini 公司发行的稳定币Gemini Dollar，与Paxos公司发行的稳定币Paxos Standard ，每个token有 1 美元支撑，旨在提供法币的稳定性，以及加密货币的速度和无国界性质。

这种受政府机构监管的稳定币的出现的意义是十分重大的。

首先，打通了数字币与法币自由流通的最后一公里。

一是解决了稳定币的币值信任问题。 USDT最大的问题就是Tether公司虽然声称1USDT锚定1美元，但并没有公开银行账户，除了它自己没人知道它到底存了多少美元。所以USDT的偿付能力就成了薛定谔的猫，没人知道。而这次纽约金融服务部的监管，正好解决了这个问题，让大家放心持有GUSD这类数字币，不用担心暴雷。

二是解决了政府的合规问题。 USDT发行这么多年，一直是以公司行为作为信任基础，没有政府参与；而这次纽约金融服务部的行为，间接表示了美国政府的态度：利用加密货币不可篡改、自由流通的金融属性来继续巩固美元在全球的霸权地位 当GUSD或者PAX及其他们的后继者逐步占领加密货币市场，人们会逐渐发现，持有此类数字货币就相当于持有美元。可以想象这对我国的外汇政策冲击有多大！

其次，为什么是ETH？

ETH是除BTC以外，去中心化程度做的最好的虚拟币。 既然USDT使用BTC的omni协议，GUSD或者PAX这类稳定币也可用OMNI协议。但BTC网络的交易速度或者叫TPS实在太低了，不适合普通货币的流通使用。ETH就比它强很多，也足够安全。从这个角度来看，ETH作为承载此类应用的网络是最合适的。（EOS也不合适，因为他不够去中心化）

可以看出，稳定币可以算作是ETH网络上的杀手级应用了，这对于包括ETH在内的各大虚拟货币，都将是好消息。

寒冬将至

发表于 2018-09-10 | 分类于 区块链 |

大饼又闪崩了！随着前两天btc在短短几分钟内就从7300跌到6200，这波反弹也随之宣告结束了。主流币种中，eth最弱，dash和btc相对强势，eos呈随波逐流的态势。币圈的信仰又一次在经受考验。

随着狂人的公众号正式停更，这次的寒冬或许已进入数九寒天，至于是几九，只有天知道。很多人黯然神伤的离开了，对虚拟币的信仰也开始了质疑，最主流的说法是：eth要归零了！大饼要破新低了！EOS超发太多还得跌！我自己也很难受，毕竟人要生活，每天都有支出，只能不得不卖币。但我的信仰却越来越坚定了。

我曾经不止一次的在想：如果我是在今年才接触的比特币，我还会像去年那样去买它吗？每次的结果都是：一定会！很明显，比特币的价值和人们的共识水平呈正比，认识他、知道他的人越多，他就越有价值！至于现在的价格是否太贵？这不是我能够回答的问题，我只知道，比特币未来的价值是不断上升的，这个趋势是长期的、不可逆的、是持续不断的。

说完了比特币，我想再说一下以太坊。最近，一大波人叫嚣ETH要归零，主要理由是：ICO黄了、竞争对手强了（EOS）等等。但我觉得，这些理由只会削弱ETH的价值，但ETH永远都不会归零！作为虚拟货币的二号选手，无数的矿工在用真金白银在支持他，上千万的活跃地址、几乎所有主流交易所都将ETH作为除BTC以外的主要交易对，这些都是ETH的价值支撑所在。一个新兴的虚拟币想要代替ETH现在的地位其难度是非常大的。即使eth的ICO属性被削弱了，但其货币属性还在，它的价值就还在！如果像狗狗币、莱特币这样的纯山寨币都没死，以太坊有什么理由会死呢？

最后，按照惯例，我要再说一下EOS。随着最近走势的变弱，我可能要修正一下之前的观点：EOS的价格在目前的熊市状态下，想要一枝独秀，几乎是不可能的！但如果熬过了熊市，那它很有可能是第一个一飞冲天的币种，毕竟天时地利人和他全占了，就差时间了！

熊市还在继续，你的信仰还在吗？

发表于 2018-08-22 | 分类于 区块链 |

币圈的熊市已经过去大半年了，前两天随着比特币从5900美金一路反弹到6600美金，好多人似乎看到了熊市终结的曙光。谁知就在昨晚就又经历了一轮暴跌，继续回踩至6400美金附近，我看很多人都渐渐的丧失信仰了。尤其是这次EOS并没有太大的反弹动作，导致很多人看衰EOS。但我想问一句：现在的币价真的就那么重要吗？

金融界的常识：价格围绕价值波动，虽然有时会偏离，但总有一天会回归。那好，我就来分析分析比特币、以太坊、EOS的价值到底有什么变化？难道就因为币价跌跌不休，他们真的就不值钱了吗？

先说说比特币，相信大家现在对比特币不会归零这个事实都没有质疑吧？目前的走势，除了空头在熊市中兴风作浪以外，主要的一点，就是美国SEC的政策影响。最近有3到4家证券公司申请的比特币ETF将得到审批结果，通过，大牛来临。不通过，也就猛跌一阵，因为比特币ETF早晚会被通过，只是时间问题。

以太坊的情况就不同了。2017年的大牛市，主要是ICO的兴起，各路众筹者将以太坊的价格不断推高。现在ICO几乎已经凉凉，币价惨淡，空气币归零等诸多因素，导致了筹到款项的人不断砸盘套现，把以太坊砸出了超级大底。但以太坊同样不会死，因为：

  1. 有共识，以太坊作为全球第2大币种，全球认可它的人数可以说是仅次于比特币。经过几年的运行，安全性也有保障。
  2. 去中心化，以太坊可以说是除比特币以外去中心化最彻底的。甚至某种意义上讲比比特币还要去中心化（没有比特大陆对矿机的把持），所以，这一点也是吸引币圈广大人员相信以太坊的一大原因。有一些人就只信任POW的币种。
  3. 未来发展，虽然V神的未来发展计划十分庞大，难度很高。但大家对他的开发能力毫不怀疑，只是要看社区的共识了，未来由Pow转向POS的路不太好走。但一旦成功，以太坊无疑将迎来第二春。

EOS目前来看，依然是币圈最具活力的，不管是技术上还是社区运营上。虽然很多人开始怀疑，甚至套现离场。但我从它2个月的迅猛发展来看，EOS的发展速度是其他项目的几倍甚至是几十倍。如果它不能成功，还有谁能成功呢？

初上手BitMEX的感受

发表于 2018-08-15 | 分类于 区块链 |

熊市已经持续8个月了，很多人已经快扛不住了。我看着自己不断缩水的账户，表面平静如水，实际也很着急。总不能这么被动吧？是否可以尝试在市场下跌中赚钱呢？那只有去做期货了。

期货市场是典型的负和市场，它会放大交易员的错误，当然也会放大收益，但期货市场正确的用法则是套期保值，与现货市场配合使用。之前我只单边做多现货，但现在，我决定尝试配合使用期货来保值了。

虚拟货币中的期货市场，可用的不多，BitMex是最老牌、最稳定，最不会坑人的一个。所以就从BitMEX入手。一开始熟悉规则的时候，自己不敢用实盘，所以用了模拟盘操作。一上手，先来100倍和50倍杠杆，还没坚持10分钟就全部爆仓，做多做空都不行。后来，调整为25倍以下杠杆才好一些，单子能够存活挺久时间。

本来，我认为在模拟盘上如果输光了，那我就不搞期货了，毕竟风险太大。可随着对盘面的熟悉，自己下单开始有了一点章法和逻辑，竟然在不断的赚钱:(，钱越搞越多了。

仔细复盘后，我发现，赚钱的主要原因是：我对模拟盘的钱不在乎。所以下单很放松，认错也及时，单子也能拿的住，对趋势没有任何幻想，顺势而为，自然就会赚钱。如果是实盘，里面的钱是我的全副身家的话，或许我就会患得患失了，那就离亏钱不远了。

如果想要做实盘，首先，要拿出你真正能够亏得起的钱来玩。每次下单量不能超过总量的一半（这个可以自己动态掌握），给自己2次犯错的机会，如果连续3次犯错，那就暂时别做了，看看自己的逻辑是否有问题。还有就是，要等到市场真正的时机到来时再下单，其他时候只看盘休息，别乱操作。你不可能挣到市场上的每一笔钱。

最后，祝大家好运！

在EOS上开发个DAPP到底有多难？

发表于 2018-08-07 | 分类于 区块链 |

最近的金融市场真的是哀鸿遍野, 大A股跌破2700点, 币圈更是跌跌不休,普遍距离1月份的最高点跌去70%左右. 熊市当中, 我们需要继续充值信仰:) 目前号称区块链3.0的EOS, 成了目前币圈最能折腾的项目:6月15日主网上线以来,RAM刚刚爆炒一轮,从高点0.9跌到0.15,套死无数小散. 现在BM又提出把计算资源打包为REX, 购买REX可以得到SEOS, 按照BM的说法:

“REX账目中的每个EOS都可以产生一个抵押EOS代币(SEOS). SEOS和EOS被存放在Bancor Relay的两个连接中,最开始的时候,EOS和SEOS对应的数量比是1:1, 买卖者可以支付一小笔交易费(如0.5%)来买卖SEOS.”

可能你和我一样,对这些规则是一脸懵逼. 结合之前炒作TPS和RAM的前科,我们很怀疑这是不是BM对EOS的新一轮炒作行为. 当然,BM的出发点是好的,就是为了降低开发者的开发成本. 那么我们就算一下,现在开发一个DAPP到底要多少成本?

一是知识成本. EOS.IO是用C++开发的,同样,基于EOS的DAPP也需要用C++来开发. 我想就这一点,就已经挡住不少人的热情了,毕竟,C++号称是最难掌握的编程语言.(反正我是不打算去学,主要是没这个时间)

二是资金成本. EOS不是号称转账免费吗? 确实是,EOS转账确实免费,而且速度非常快. 但是, 新建一个EOS账号是要钱的! 一开始,新建一个账号最少要消耗1.5EOS.如果按照EOS50元一枚的价格算,建立一个账号最少要花80元左右.而且,新建EOS账号的成本是随着RAM成本的升高而升高的, 在RAM价格高达0.9左右的时候,新建一个账号要花200块钱!

这还仅仅是开一个账号,做开发,还要购买大量的RAM和CPU来保证合约的正常运行. 以发行一个自有代币举例: 发代币,你的账号起码要有200KB的内存,目前内存价格是0.178EOS/KB,也就是说,你要花至少36个EOS才能开始下一步操作.(我刚刚买了200KB内存用来发币,发了两千一百万枚NBT,谁想要留下账户给我,我转给你一些拿去玩)

光买内存还不行,还得要足够的CPU和带宽资源,虽然现在CPU不需要购买,只需要抵押. 但随着BM宣称的REX的到来,未来所有的资源都可以拿来炒作了. 现在CPU的抵押价格已经暴涨50倍了!

不过,从好的方面来看, BM的种种做法提高了EOS整个生态的活力,同时让投机者和开发者各取所需,确实是一大创新.希望REX的出现能够实实在在的降低开发者的成本.

从开发难度来看,目前,EOS的开发者依然十分活跃. 已经有开发者做出了基于EOS的开发工具,例如FIBOS,可以用JavaScript来进行DAPP开发(需要了解的同学加我微信).

EOS的未来还是充满光明的,当前要紧的是,抓紧时间学开发!
